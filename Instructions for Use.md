Unless otherwise specified in the future, 
all materials in this repository support Minecraft: Java 1.17. 
By using them, 
you agree to the terms outlined in the README.txt within the texture pack and the open-source license of this repository.
The Yu's Golden Apple.zip texture pack is not compatible with the Yu_Purity's Totem.zip texture pack. 
For compatibility, please download Yu's Golden Apple v2.zip. 
For commercial use, please obtain the author's permission.
This texture allows you to change it to your own Minecraft skin. 
You can replace the Minecraft skin textures in the "assets/Minecraft/texture/item/*" directory with your own skin textures, 
but do not change the file names, otherwise it will not take effect and may cause texture errors.
